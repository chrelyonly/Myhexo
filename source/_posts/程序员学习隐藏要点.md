## java

### 数组集合不能用 !=null 来判断
### 数组集合为空的情况下,会返回 []  而不是 "空"
``` bash
$ List<string>  list
$ list!=null
$ []
```

## javascript

### var 变量它的值是什么类型的,它就是什么类型的.
### 它拥有隐藏条件转化,不需要手动转
``` bash
$ var t=1  数字类型
$ var t="哈哈"  string类型
$ var t=    
```
